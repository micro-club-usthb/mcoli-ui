Welcome to **mc-ui**, the official component registry for Micro Club. mc-ui provides pre-built themes and components that extend [shadcn/ui](https://ui.shadcn.com) with custom styling.

> **Important:** mc-ui is not a standalone component library. It is a **theme and component registry** that requires shadcn/ui as a prerequisite. Think of it as an extension that adds Micro Club's custom themes and components to your existing shadcn/ui project.

## Prerequisites

mc-ui requires an existing **shadcn/ui** project. If you don't have one, you must set it up first.

> **Warning:** mc-ui is built on top of shadcn/ui. You **must** initialize shadcn/ui before using mc-ui.
> 
> Follow the [shadcn/ui installation guide](https://ui.shadcn.com/docs/installation) to set up shadcn/ui in your project.

## Initialize mc-ui

Once shadcn/ui is set up, you can initialize mc-ui to add Micro Club's themes:

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab>
    ```sh
npx mc-ui@latest init
    ```
  </Tab>
  <Tab>
    ```sh
pnpm dlx mc-ui@latest init
    ```
  </Tab>
  <Tab>
    ```sh
yarn dlx mc-ui@latest init
    ```
  </Tab>
  <Tab>
    ```sh
bunx mc-ui@latest init
    ```
  </Tab>
</Tabs>

The `init` command will prompt you to select a theme for your project.

Alternatively, you can install the CLI globally:

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab>
    ```sh
npm install -g mc-ui@latest
mc-ui init
    ```
  </Tab>
  <Tab>
    ```sh
pnpm add -g mc-ui@latest
mc-ui init
    ```
  </Tab>
  <Tab>
    ```sh
yarn global add mc-ui@latest
mc-ui init
    ```
  </Tab>
  <Tab>
    ```sh
bun add -g mc-ui@latest
mc-ui init
    ```
  </Tab>
</Tabs>

## Available Themes

mc-ui ships with five pre-built themes:

| Theme | Description |
|-------|-------------|
| **Primary** | A professional blue primary with purple secondary accents |
| **Secondary** | A purple primary with blue secondary accents |
| **GameDev** | A vibrant pink primary with yellow accents for gaming |
| **Robotics** | A technical blue primary with accent blue for precision |
| **IT** | A clean green primary with gray accents for IT |

## Why mc-ui?

- **Copy & Paste**: Own your code. Components are copied directly into your project.
- **Themeable**: Pre-built themes that integrate seamlessly with shadcn/ui.
- **Custom Components**: Micro Club-specific components built on Radix UI primitives.
- **Type-Safe**: Written in TypeScript for better developer experience.
