Welcome to **mcoli-ui**, the official component registry for Micro Club. mcoli-ui provides pre-built themes and components that extend [shadcn/ui](https://ui.shadcn.com) with custom styling.

> **Important:** mcoli-ui is not a standalone component library. It is a **theme and component registry** that requires shadcn/ui as a prerequisite. Think of it as an extension that adds Micro Club's custom themes and components to your existing shadcn/ui project.

## Prerequisites

mcoli-ui requires an existing **shadcn/ui** project. If you don't have one, you must set it up first.

> **Warning:** mcoli-ui is built on top of shadcn/ui. You **must** initialize shadcn/ui before using mcoli-ui.
> 
> Follow the [shadcn/ui installation guide](https://ui.shadcn.com/docs/installation) to set up shadcn/ui in your project.

## Available Themes

mcoli-ui ships with five pre-built themes:

| Theme | Description |
|-------|-------------|
| **primary** | A professional blue primary with purple secondary accents |
| **secondary** | A purple primary with blue secondary accents |
| **game-dev** | A vibrant pink primary with yellow accents for gaming |
| **robotics** | A technical blue primary with accent blue for precision |
| **it** | A clean green primary with gray accents for IT |

## Initialize mcoli-ui

Once shadcn/ui is set up, you can initialize mcoli-ui to add Micro Club's themes.

### Interactive Mode

The `init` command will prompt you to select a theme with a visual color picker:

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab>
    ```sh
npx mcoli-ui@latest init
    ```
  </Tab>
  <Tab>
    ```sh
pnpm dlx mcoli-ui@latest init
    ```
  </Tab>
  <Tab>
    ```sh
yarn dlx mcoli-ui@latest init
    ```
  </Tab>
  <Tab>
    ```sh
bunx mcoli-ui@latest init
    ```
  </Tab>
</Tabs>

### Direct Theme Selection

You can also specify the theme directly as an argument:

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab>
    ```sh
npx mcoli-ui@latest init primary
    ```
  </Tab>
  <Tab>
    ```sh
pnpm dlx mcoli-ui@latest init primary
    ```
  </Tab>
  <Tab>
    ```sh
yarn dlx mcoli-ui@latest init primary
    ```
  </Tab>
  <Tab>
    ```sh
bunx mcoli-ui@latest init primary
    ```
  </Tab>
</Tabs>

Replace `primary` with any available theme: `primary`, `secondary`, `game-dev`, `robotics`, or `it`.

### Global CLI Installation

Alternatively, you can install the CLI globally:

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab>
    ```sh
npm install -g mcoli-ui@latest
mcoli-ui init primary
    ```
  </Tab>
  <Tab>
    ```sh
pnpm add -g mcoli-ui@latest
mcoli-ui init primary
    ```
  </Tab>
  <Tab>
    ```sh
yarn global add mcoli-ui@latest
mcoli-ui init primary
    ```
  </Tab>
  <Tab>
    ```sh
bun add -g mcoli-ui@latest
mcoli-ui init primary
    ```
  </Tab>
</Tabs>

## Why mcoli-ui?

- **Copy & Paste**: Own your code. Components are copied directly into your project.
- **Themeable**: Pre-built themes that integrate seamlessly with shadcn/ui.
- **Custom Components**: Micro Club-specific components built on Radix UI primitives.
- **Type-Safe**: Written in TypeScript for better developer experience.
