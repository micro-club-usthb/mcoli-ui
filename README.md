# MC UI – Micro Club UI Components Library

MC UI is the official React UI component library for Micro Club, providing reusable, accessible, and customizable components for building modern web applications. The project is built with Next.js and shadcn/ui, and uses Fumadocs to showcase and document the components.

## Features

- **Reusable Components:** Consistent, production-ready React components.
- **Accessibility:** Components follow best practices for accessibility.
- **Customizable:** Easily adapt components to your design requirements.
- **Modern Tooling:** Built with Next.js, shadcn/ui, and documented using Fumadocs.

## Getting Started

To start the development server:

```sh
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser to view the documentation and component showcase.

## Scripts

- **Build Component Registry:**  
  Compiles all MC UI components into a JSON registry compatible with shadcn/ui CLI and v0.

  ```sh
  npm run build:registry
  ```

- **Build Documentation:**  
  Generates the documentation site using Fumadocs.

  ```sh
  npm run build:docs
  ```

## Documentation

Documentation and live component demos are powered by Fumadocs. You can customize documentation options (such as frontmatter schema) via the included `source.config.ts` file. For more details, see the [Fumadocs Introduction](https://fumadocs.dev/docs/mdx).
